# Bug Report 3

## Title
Profile image upload validation is not clearly implemented

## Severity
Medium

## Observation

The application supports profile image uploads, but validation rules for file type and file size are not clearly defined.

## Expected Result

The system should accept only supported image formats (JPG, JPEG, PNG) and enforce file size restrictions.

## Potential Impact

Invalid or oversized files may cause storage, security, or display issues.
